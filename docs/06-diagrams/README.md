# Diagrams

This section contains visual representations of system architecture, flows, and data structures.

## Structure

- **architecture/**: System architecture diagrams
- **flows/**: Process and algorithm flow diagrams
- **data/**: Data model and entity relationship diagrams

## Purpose

Visual documentation:
- Aids understanding of complex systems
- Facilitates communication
- Supports onboarding
- Enables quick reference

## Formats

Diagrams can be created using:
- Mermaid (markdown-compatible)
- PlantUML
- Draw.io
- Or any preferred diagramming tool
